### JAVA College 3rd Sem

## **Commands :**

Create a Directory :

```
mkdir directory_name
```

Create File :

```
pluma fileName
```

Change Directory :

```
cd /path/to/directory
```

Compile Java :

```
javac FileName.java
```

Run :

```
java FileName
```
